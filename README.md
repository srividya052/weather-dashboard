# Weather Dashboard

A responsive weather dashboard built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API.

# Live Demo
https://srividya052.github.io/weather-dashboard/

## Features
- Search weather by city name
- Display temperature, humidity, and wind speed
- Show weather descriptions and icons
- Generate forecast-based weather advisories
- Loading states and error handling
- Press Enter to search
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- OpenWeatherMap Current Weather API
- OpenWeatherMap 5-Day Forecast API

## Getting Started

1. Clone this repository:

git clone https://github.com/srividya052/weather-dashboard.git


2. Add your OpenWeatherMap API key in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

3. Open `index.html` in your browser.
